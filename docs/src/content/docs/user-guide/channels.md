채널은 외부 메시징 플랫폼과 OpenClaw을 연결하는 인터페이스다. 15개 이상의 플랫폼을 지원한다.

- [채널 개요](https://docs.openclaw.ai/channels) — 지원 채널 목록과 공통 개념

## 메시징 플랫폼

| 플랫폼 | 공식 문서 | 프로토콜 |
|--------|----------|---------|
| WhatsApp | [WhatsApp](https://docs.openclaw.ai/channels/whatsapp) | WhatsApp Business API |
| Telegram | [Telegram](https://docs.openclaw.ai/channels/telegram) | Bot API (Grammy) |
| Discord | [Discord](https://docs.openclaw.ai/channels/discord) | Discord.js |
| IRC | [IRC](https://docs.openclaw.ai/channels/irc) | IRC 프로토콜 |
| Slack | [Slack](https://docs.openclaw.ai/channels/slack) | Socket Mode / HTTP Events API |
| Feishu (라크) | [Feishu](https://docs.openclaw.ai/channels/feishu) | Feishu Open API |
| Google Chat | [Google Chat](https://docs.openclaw.ai/channels/googlechat) | Google Chat API |
| Mattermost | [Mattermost](https://docs.openclaw.ai/channels/mattermost) | Mattermost API |
| Signal | [Signal](https://docs.openclaw.ai/channels/signal) | Signal CLI |
| iMessage | [iMessage](https://docs.openclaw.ai/channels/imessage) | AppleScript (macOS) |
| MS Teams | [MS Teams](https://docs.openclaw.ai/channels/msteams) | Bot Framework |
| LINE | [LINE](https://docs.openclaw.ai/channels/line) | LINE Messaging API |
| Matrix | [Matrix](https://docs.openclaw.ai/channels/matrix) | Matrix Protocol |
| Zalo | [Zalo](https://docs.openclaw.ai/channels/zalo) | Zalo OA API |
| Zalo User | [Zalo User](https://docs.openclaw.ai/channels/zalouser) | Zalo User API |

## 채널 설정

| 항목 | 공식 문서 | 설명 |
|------|----------|------|
| 페어링 | [Pairing](https://docs.openclaw.ai/channels/pairing) | 채널 계정과 게이트웨이 연결 |
| 그룹 메시지 | [Group Messages](https://docs.openclaw.ai/channels/group-messages) | 그룹 채팅 처리 방식 |
| 그룹 관리 | [Groups](https://docs.openclaw.ai/channels/groups) | 그룹 정책과 허용목록 |
| 브로드캐스트 그룹 | [Broadcast Groups](https://docs.openclaw.ai/channels/broadcast-groups) | 방송형 그룹 설정 |
| 채널 라우팅 | [Channel Routing](https://docs.openclaw.ai/channels/channel-routing) | 멀티채널 라우팅 규칙 |
| 위치 정보 | [Location](https://docs.openclaw.ai/channels/location) | 위치 기반 기능 |
| 트러블슈팅 | [Troubleshooting](https://docs.openclaw.ai/channels/troubleshooting) | 채널 연결 문제 해결 |

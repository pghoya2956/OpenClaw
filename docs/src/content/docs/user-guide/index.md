이 섹션은 [OpenClaw 공식 문서](https://docs.openclaw.ai)의 내용을 주제별로 정리한다. 공식 문서는 **"어떻게 설정하고 사용하는가"**에 초점이 있으며, 이 가이드는 공식 문서의 방대한 페이지를 한국어로 요약하고 빠르게 찾을 수 있도록 구성했다.

## 공식 문서와의 관계

| 구분 | 공식 문서 | 이 사이트의 Internals |
|------|----------|---------------------|
| 초점 | 설치, 설정, 사용법 | 소스 코드 내부 동작 |
| 대상 | 운영자, 사용자 | 개발자, 기여자 |
| 위치 | [docs.openclaw.ai](https://docs.openclaw.ai) | 이 사이트의 "시작하기" 이후 섹션 |

- **[도구 & 스킬](/user-guide/tools/)** — 내장 도구, 브라우저, 스킬, 플러그인
- **[모델 & 프로바이더](/user-guide/models/)** — LLM 프로바이더 설정, 모델 페일오버

- **[게이트웨이 운영](/user-guide/gateway/)** — 설정, 보안, 프로토콜, 원격 접속, 웹 UI
